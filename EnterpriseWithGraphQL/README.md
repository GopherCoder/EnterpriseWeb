# Lottery

> 小程序：抽奖助手


## 抽象实体

- 用户账号
- 抽奖
    - 发起抽奖
        - 不同类型
        - 奖项
            - 名称
            - 份额
            - 级别：默认一等
        - 开奖时间（5天之内，创建时间节点之后）
        - 开奖条件
            - 类型
            - 条件
                - 按时间自动开奖
                - 按人数自动开奖
                - 即开即中
- 查询
    - 发起
    - 参与        
- 首页：推荐的抽奖
